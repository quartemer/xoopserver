XOOPS Cube Package Installation

1. Télécharger une distribution de XOOPS Cube

2. Décompresser les fichiers dans : W:/xoopserver/www/nomDistribution/

3. Installer depuis votre navigateur : http://localhost/nomDistribution/

Note: Nom d'utilisateur de la base de données : root (sans mot-de-passe)
